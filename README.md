## ğŸ¨ Customization

### Colors
Edit CSS variables in `style.css`:
```css
:root {
    --primary-color: #6C63FF;
    --secondary-color: #FF6584;
    --accent-color: #00D9FF;
    --dark-bg: #0F0F1E;
}
```

### Content
- Update `index.html` for main content
- Edit `resume.md` for resume text
- Modify `script.js` for functionality

### Profile Photo
- Replace `profile.jpg` with your photo
- Recommended size: 400x400px
- Format: JPG or PNG
